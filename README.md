# Mindvalley Manifesting Hub

A beautiful, interactive landing page for Mindvalley's Free Manifesting Masterclasses.

## Features

- Interactive quiz to match users with the perfect manifesting masterclass
- Responsive design optimized for all devices
- Modern UI with Mindvalley design system
- Personalized recommendations based on user responses

## Technologies

- Pure HTML, CSS, and JavaScript
- No dependencies required
- Google Fonts (Inter, Gilroy)

## Setup

Simply open `index.html` in a web browser or deploy to any static hosting service.

## GitHub Pages

This repository is configured for GitHub Pages. Once pushed to GitHub, enable GitHub Pages in the repository settings to publish the site.
